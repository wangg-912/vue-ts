# vue-ts

## 基于 typescript 3.4.3、Vue Cli3.x、axios、element-ui 为基础项。实现国际化，全|缩屏，全局控件大小等系统管理平台。

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Run your tests

```
yarn run test
```

### Lints and fixes files

```
yarn run lint
```

### Run your unit tests

```
yarn run test:unit
```
